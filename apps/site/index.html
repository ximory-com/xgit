随时随地，轻松管理你的 GitHub 仓库 Manage your GitHub repos, anywhere

XGit 是一款专为移动端设计的轻量级 GitHub 客户端，让你在手机上也能流畅地查看、编辑与提交代码。 XGit is a lightweight GitHub client designed for mobile. Browse, edit and commit on the go.

基础功能 Core features

仓库列表 Repository list 规划中Planned

登录后展示我的仓库，支持搜索与最近活跃排序。 Show repos after sign-in, with search and recent-activity sort.

目录浏览 Directory browsing 规划中Planned

逐层进入（移动端单屏），桌面端三栏回退。 Stacked single-screen on mobile; 3-pane navigation on desktop.

文件查看/编辑（文本） View / edit (text) 规划中Planned

基础编辑可用；后续切换 Monaco 编辑器。 Basic editor first; switch to Monaco later.

多文件勾选一次提交 Multi-file commit 规划中Planned

同时暂存多个文件，填写一次提交信息。 Stage multiple files and commit once.

Diff 对比 Diff 规划中Planned

文件级 Diff，后续支持行内高亮。 File-level diff; inline highlights later.

ZIP 上传/下载 ZIP upload / download 规划中Planned

支持打包下载目录，上传 ZIP 并解包入仓。 Download folders as ZIP; upload ZIP and unpack into repo.

FAQ

需要服务器吗？Do I need a server?
不需要，全部前端完成，直接对接 GitHub API。No. Everything runs in the browser with GitHub API.
支持私有仓库吗？Private repos?
支持，只需授予 Token（建议 scope: repo）。Yes. Grant a token (scope: repo) and you’re good.
移动端体验如何？Mobile experience?
采用单屏栈式导航，操作更顺手。Single-screen stack navigation for smooth operations.
