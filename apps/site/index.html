产品定位

只把 仓库相关操作 做到极致：浏览 / 编辑 / Diff / 多文件一次提交 / 历史 / 分支 / 上传下载。不做 Issues/PR 等复杂协作（必要时仅跳转）。

FAQ

  • 需要服务器吗？ 不需要，全部在前端完成，直接对接 GitHub API。
  • 支持私有仓库吗？ 支持，只需授权 Token（建议 scope: repo）。
  • 移动端体验如何？ 采用单屏栈式导航（逐层进入），操作更顺手。
